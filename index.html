1.University Project/Personal Project

Song Recommendation Based on Emotion Mood Detection

This project leverages emotion detection from facial expressions to enhance music recommendation systems. The model first detects faces in both still images and videos, analyzing facial features to predict the user's emotional state. Based on the identified emotion, the system then processes a Spotify dataset from a Kaggle contest to generate personalized song recommendations that match the user's mood. This approach combines computer vision techniques for emotion recognition with music data to provide a dynamic, mood-driven music experience.
